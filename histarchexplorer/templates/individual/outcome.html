Software

OpenAtlas

OpenAtlas is an open source database software developed especially to acquire, edit and manage research data from various fields of humanities like history, archaeology and cultural heritage as well as related scientific data. Within the RELIC Project various features have been developed to facilitate the use of OpenAtlas for the project.

THANADOS

THANADOS is an open research infrastructure for the study of medieval and early modern burial archaeology. It integrates archaeological, anthropological and spatial data to enable comparative analysis, long-term preservation and open access to burial-related cultural heritage data.

Project Site ↗

THANADOS Next Generation

The next generation THANADOS infrastructure (HistArchExplorer) is developed within the RELIC project and expands the platform with enhanced spatial analysis tools, improved data integration workflows and modern visualization approaches for archaeological and historical research data.

Repository ↗

Datasets

RELIC / THANADOS API

The THANADOS API provides structured access to archaeological, anthropological and spatial burial data generated within the RELIC research environment. It supports interoperable data exchange, reproducible research workflows and integration into spatial humanities applications.
